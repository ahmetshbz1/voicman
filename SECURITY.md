# Security Policy | Güvenlik Politikası

## English

If you discover a security issue in Voicman, please do not open a public issue first.

Instead, share:

- a clear description of the issue
- impact
- reproduction steps
- affected macOS version
- logs or screenshots if relevant

Please use a private disclosure channel controlled by the maintainers. If no dedicated channel is published yet, contact the repository owner directly and wait for triage before public disclosure.

Security-sensitive areas in this project include:

- microphone access
- speech recognition authorization
- accessibility-based paste automation
- clipboard handling
- global hotkey registration

## Türkçe

Voicman’da bir güvenlik açığı keşfedersen lütfen önce public issue açma.

Bunun yerine şunları paylaş:

- açığın net açıklaması
- etkisi
- yeniden üretme adımları
- etkilenen macOS sürümü
- varsa log veya ekran görüntüsü

Maintainer’ların kontrol ettiği özel bir bildirim kanalını kullan. Henüz ayrı bir kanal yayımlanmadıysa repo sahibiyle doğrudan iletişime geç ve public disclosure öncesi triage sürecini bekle.

Bu projede güvenlik açısından hassas alanlar:

- mikrofon erişimi
- konuşma tanıma yetkilendirmesi
- erişilebilirlik tabanlı otomatik yapıştırma
- pano yönetimi
- global hotkey kaydı
